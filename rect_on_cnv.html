Overview

Grid is drawn on the canvas. You have to press the mouse button and drag the mouse to the destination
point.

Coding

Drawing simple canvas

First three line includes the required module in the program. iup.canvas create a canvas.
iup.dialog create dialog. Here vbox represents a vertical box. We put canvas into a vertical box.
dg:showxy(iup.CENTER,iup.CENTER) shows the dialog. last three line use to be able to run
this script inside another context.

Drawing grid on canvas

function drawGrid will draw a grid on the canvas. function map_cb create a cd canvas on which
we can draw. function unmap_cb kill the cd canvas.

Fit the rectangle position to grid

This function(check_gird) takes four variable as input and manipulates them, and set the input
variable best fit to the grid.

Drawing rectangle on canvas

There are three functions. First one drawRect simply draw the grid on the canvas after that check the coordinate and draw
a rectangle on the canvas. Second function cnv:button_cb call when a mouse button is pressed it set the starting coordinates
of a rectangle. Third function cnv:motion_cb call when the mouse is moved and set the end point of the rectangle.
